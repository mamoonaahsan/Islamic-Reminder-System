# 🌙 Islamic Reminder System (C++)

A console-based Islamic Reminder System developed in C++ using Object-Oriented Programming concepts.

## Features

- 🕌 Prayer Reminder
- 🕒 Prayer Timetable
- 📖 Hadith Reminder
- 📜 Quran Ayat with Translation
- 🤲 Dua Reminder
- 🧭 Qibla Direction
- 📅 Hijri Calendar Conversion
- 💰 Zakat Calculator
- 📿 Tasbeeh Counter

## Concepts Used

- Classes and Objects
- Inheritance
- Polymorphism
- Virtual Functions
- Dynamic Memory Allocation
- Arrays
- String Handling

## Language

C++
